Stripe Integration Help

To enable Stripe auto-renewal:

  1. Go to Stripe Webhooks
  2. Add a new webhook endpoint
  3. Set URL to: https://your-domain.com/api/stripe/webhook
  4. Select events: customer.subscription.updated, customer.subscription.deleted, invoice.payment_succeeded, invoice.payment_failed
  5. Copy the signing secret and paste above

When a subscription is updated, renewed, or cancelled in Stripe, the license status will be automatically synchronized. Auto-renewal licenses will create Stripe checkout sessions for payment.
